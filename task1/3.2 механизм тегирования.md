| **Тип данных**                              | **Механизм тегирования**                                                               | **Применение инструментов**                                                                  |
| ------------------------------------------- | -------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------- |
| **Персональные данные (PII)**               | Тег "confidential" для всех персональных данных, "PII" для идентифицируемой информации | Использование базы данных с метками для контроля безопасности и доступа                      |
| **Медицинские данные (PHI)**                | Тег "medical" для всех медицинских данных, "PHI" для личных медицинских записей        | Системы обработки данных с поддержкой тегирования (например, в системах EHR)                 |
| **Платежные данные**                        | Тег "payment" для всех данных о платежах, "tokenized" для токенизированных данных      | Инструменты для обработки платежей с поддержкой защиты данных (например, PCI DSS)            |
| **Данные о товарно-материальных ценностях** | Тег "inventory" для всех данных о товарах и запасах                                    | Использование ERP-систем с поддержкой защиты и тегирования данных                            |
| **Данные о записях на приём**               | Тег "appointment" для данных о медицинских записях                                     | Системы для медицинского учета и записи, поддерживающие использование тегов для безопасности |

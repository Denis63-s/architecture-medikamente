| **Тип данных**                              | **Данные для защиты**                          | **Способы защиты**                                                       |
| ------------------------------------------- | ---------------------------------------------- | ------------------------------------------------------------------------ |
| **Персональные данные (PII)**               | ФИО, адрес, телефон, дата рождения             | - Шифрование (при хранении и передаче данных)                            |
|                                             |                                                | - Обфускация (например, замена части номера телефона)                    |
|                                             |                                                | - Обезличивание (для анализа, но без ФИО)                                |
| **Медицинские данные (PHI)**                | Результаты анализов, диагнозы, карты пациентов | - Шифрование данных (для защиты конфиденциальности)                      |
|                                             |                                                | - Обезличивание (в случае использования для статистики и анализа)        |
|                                             |                                                | - Псевдонимизация (замена имени пациента на псевдоним)                   |
| **Платежные данные**                        | Номера карт, данные о транзакциях              | - Токенизация (замена реальных данных на токены)                         |
|                                             |                                                | - Шифрование (для защиты данных при передаче и хранении)                 |
| **Данные о товарно-материальных ценностях** | Инвентаризация, поставки, цены                 | - Шифрование (для хранения данных о товарных ценностях)                  |
|                                             |                                                | - Контроль доступа (ограничение прав доступа к этим данным)              |
| **Данные о записях на приём**               | Данные о записях (пациенты, специалисты, даты) | - Шифрование (при передаче и хранении записей)                           |
|                                             |                                                | - Обфускация (для защиты частичной информации, например, времени записи) |

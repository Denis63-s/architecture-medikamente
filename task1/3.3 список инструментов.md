| **Процесс/Тип данных**                      | **Инструмент**                                            | **Методы защиты и меры безопасности**                                                  |
| ------------------------------------------- | --------------------------------------------------------- | -------------------------------------------------------------------------------------- |
| **Обработка персональных данных (PII)**     | **Data Security Gateway (DSG)**                           | - Шифрование данных при передаче и хранении.                                           |
|                                             | **RBAC/ABAC (Role-Based/Attribute-Based Access Control)** | - Контроль доступа на основе ролей/атрибутов.                                          |
|                                             | **Audit & Compliance Layer**                              | - Аудит действий и операций с конфиденциальными данными.                               |
| **Медицинские данные (PHI)**                | **Шифрование (AES-256, TLS)**                             | - Защита всех медицинских данных через шифрование.                                     |
|                                             | **Псевдонимизация и обезличивание**                       | - Применение псевдонимизации для анализа и отчетности.                                 |
|                                             | **Система защиты данных**                                 | - Разделение данных на категории с защитой и отдельными правами доступа.               |
| **Платежные данные**                        | **Токенизация**                                           | - Применение токенизации для замены реальных данных на токены.                         |
|                                             | **PCI DSS**                                               | - Соблюдение стандартов безопасности для обработки платежей.                           |
| **Данные о товарно-материальных ценностях** | **ERP-системы (например, 1C)**                            | - Внедрение шифрования и контроля доступа к данным о товарах.                          |
|                                             | **Мониторинг и аудит**                                    | - Постоянный мониторинг доступа и операций с ТМЦ.                                      |
| **Данные о записях на приём**               | **Мобильное приложение для пациента**                     | - Шифрование данных при передаче через мобильные приложения и веб-интерфейсы.          |
|                                             | **Система записи и аудита (например, через CRM)**         | - Интеграция с системой мониторинга, которая отслеживает доступ и изменения в записях. |
